# Chemora 🧪

Chemora is an all-in-one chemistry toolkit to solve chemical problems easily.

## Features
- Molar Mass Calculator
- Dynamic element input
- Real-time calculation

## Tech Stack
- React
- TypeScript
- Tailwind CSS

## Future Plans
- Equation Balancer
- Periodic Table
- Formula Parser

## Getting Started
```bash
npm install
npm run dev
